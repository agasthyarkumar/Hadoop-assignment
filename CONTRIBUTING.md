Thank you for contributing!

Guidelines
- Fork the repository and create a feature branch for your work.
- Keep changes small and focused; prefer a single concern per PR.
- Follow existing Java style in the repo.
- If you add tests, include instructions to run them.

Code review
- Open a PR against `main` and include a brief description of the change.
- CI/tests (if any) should pass before merging.

License
- By contributing you agree to license your contributions under the same MIT license as this project.
